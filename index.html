Introducing Pico.

A supidity simple & blazing fast, flat file CMS

Making the web easy.

Stupidly Simple

Picos makes creating and maintaing a website as siple as editing text files.

Blazing Fast

Picos is seriously lightweight and doesn't use a database, making it super fast.

No Database

Picos is a "flat file" CMS. meaning no database woe's, no MySQL queries, nothing

Markdown Formatting

Edit your website in your favourite text editor usig simple markdown formatiing

Twig Templates

Pico using the Twig templating engine, for powerfull and flxibale themes.

Open Source

Pico is completely free and open source, released under the MIT license.
